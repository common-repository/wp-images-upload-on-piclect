== Description ==

Quick and easy, your site will not load. Give your WordPress site image upload feature. 
This plugin uses piclect.com site. And uploads of photos on the page to link to your site added.

Demo for wordpress website: http://makaleci.com/dersler/wordpress/php_file_cache/?page_id=546

* Jquery ajax image upload
* Multi and Drag & Drop image upload
* Admin setting pages
* Piclect.com collections check and update
* All image upload your piclect account added.
* Upload all images piclect page link your wordpress site.
* All images upload insert your wordpress db. (for next version)
* All collection check and insert your wordpress db. (for next version)
* Resize images upload
* Link bulk area after upload
* simple and quick :)

== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress. (Upload directory 'kuaza-post-shared-tracker' to the '/wp-content/plugins/' directory)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the plugin at the plugin administration page
4. Create API key for your wordpress site here: http://piclect.com/developer
5. After validate your wordpress site, insert Api key and Api secret code wordpress plugins settings page.
4. And more please look this page: (Turkish) http://makaleci.com/wordpress-sitelerinize-resim-yukleme-ozelligi-ekleyin-shortcode-piclect.html
5. English document coming soon, but easy simple :)

== Frequently Asked Questions ==

= How do I add post/page ? =

Very very easy. Insert this shortcode page/post text area:
<code>[up_piclect]</code>

= How do I add My piclect.com collections? =

Go to the wordpress plugins settings page and update settings :) 

= How do I change css style form? =

Go to the wordpress plugins settings page and change style area.
